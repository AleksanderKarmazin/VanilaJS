Basic Git commands

1. git clone https://github.com/AleksanderKarmazin/VanilaJS.git

2. git pull

3. git status

4. git add *

5. git commit -m "Git class Part 1. First commit"

-m означет что пишется коментарий к коммиту

Инициализируем новый репозиторий git init Добавляем файлы (все) git add . Если файлы не добавляются, то добавляем каждый вручную git add README.md Делаем коммит git commit -m "First commit" Пушим git push -u origin master
